# vue-web-practice
基于vue，使用stylus、pug预编译，结合element、v-chart、leaflet开发的前端实践项目。

* nodejs版本 - 10.13.0
* yarn版本 - 最新

## 安装及使用

* 在项目目录下打开终端
* ```yarn``` 安装依赖
* ```npm start``` 启动项目
